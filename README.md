# 🛍️ Shopaholic - Indian Fashion E-Commerce Platform

![Shopaholic Logo](frontend/logo.jpeg)

A fully functional e-commerce website for Indian fashion with clothing items in ₹ (Indian Rupees). Built with HTML, CSS, and JavaScript.

## ✨ Features

- 👕 **19+ Indian Clothing Items** - Men's Wear, Women's Wear, Kids, Ethnic, Footwear
- 🛒 **Working Shopping Cart** - Add/remove items, update quantities
- ❤️ **Wishlist** - Save items for later
- 🔐 **User Authentication** - Login/Signup functionality
- 💰 **Indian Rupee (₹)** - All prices in INR
- 📱 **Fully Responsive** - Works on mobile, tablet, desktop
- 🎨 **Modern UI** - Clean, vibrant design with animations
- 🏷️ **Discount Badges** - Show original vs discounted prices
- ⭐ **Product Ratings** - Star ratings and reviews count

## 🎯 How to Use

1. **Browse Products** - Scroll through the clothing collection
2. **Filter by Category** - Use the navigation tabs
3. **Sort Products** - By price low/high, name
4. **Add to Cart** - Click "Add to Cart" button
5. **Add to Wishlist** - Click heart icon
6. **View Cart** - Click on cart indicator
7. **Checkout** - Click "Checkout" and select payment method
